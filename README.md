# 🐜 Ant-Sized Adventure

Ant-Sized Adventure is a third-person 3D platformer built in Unity where you explore a giant backyard — from the perspective of a tiny character. What’s normal-sized to humans becomes an obstacle course of epic proportions when you’re ant-sized!

This is a personal solo project, created to sharpen my Unity development skills, from programming to animation and asset workflow.

## 🎮 Gameplay Summary

- Control a tiny adventurer navigating through an oversized environment  
- Use simple platforming mechanics to jump across leaves, sticks, and other natural obstacles  
- Pick up and throw basic weapons (sticks and rocks) to interact with the environment  
- Switch weapons using a UI inventory system  
- Experience smart, responsive jump animations depending on direction and terrain

## ✅ Features Implemented

- 🕹️ Third-person movement controller (custom C#)
- 🧍‍♂️ Smart jump animations:
  - Up_Jump → when jumping onto ledges  
  - Down_Jump → when jumping off platforms  
  - Neutral Jump → when jumping normally
- 🔄 Physics-based weapon pickup & throw (stick and rock)
- 🧰 Inventory system with UI buttons to equip weapons
- 🎨 Stylized low-poly art using Meshy.ai → Blender → Unity pipeline

## 🔧 Tech Stack

- Unity (URP)
- C# (Visual Studio)
- Blender (mesh editing & collider setup)
- Meshy.ai (AI-generated base models)
- Legacy Unity UI (Canvas-based)
